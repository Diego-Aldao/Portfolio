02.

proceso

En todos los proyectos, el proceso para la creación del mismo involucraba:

  • Conversión semántica de PSD a HTML5 con gran atención a los detalles.

  • Aplicación de los 3 componentes principales del Responsive Design, los cuales son: Layouts flexibles, media queries, media flexible (imagenes, videos, etc)

  • Prueba de código en varios navegadores para garantizar la compatibilidad cross-browser.

  • Uso y aprovechamiento de plugins populares para componentes interactivos comunes o complejos.
